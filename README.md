# findwordfromshuffleletters01
find Vietnamese word from shuffle letters</br>
Example input: a/S/c/ạ/m</br>
Output:</br>
am Sạc</br>
Sa cạm</br>
Sa mạc</br>
Sam cạ</br>
Sạ cam</br>
Sạc am</br>
Sạc ma</br>
Sạm ca</br>
ca Sạm</br>
cam Sạ</br>
cạ Sam</br>
cạm Sa</br>
ma Sạc</br>
mạc Sa</br>
